(constant) mimeDb :Object.<string, {source: string, extensions: Array.<string>, charset: (string|null)}>

A database of MIME types with their properties. Each MIME type entry contains:

  • source {string}: The source of the MIME type definition (e.g., "iana").
  • extensions {string[]}: An array of associated file extensions.
  • charset {string|null}: The character set used for the MIME type, or null if not applicable.

Parsed HTTP request object containing:

  • method {string}: HTTP method (e.g., "GET", "POST").
  • path {string}: Path from the request line, without the query string.
  • version {string}: HTTP version (e.g., "HTTP/1.1").
